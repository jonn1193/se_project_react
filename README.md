# WTWR

WTWR is a React app that uses weather data to recommend clothing items for the current conditions.

## Project Features

- Weather data from the OpenWeather API
- Clothing cards filtered by weather type
- Add garment modal
- Clothing item preview modal
- Responsive layout based on the project design

## Deployment

This project uses a local json-server API for clothing items. To view the current version, run the app locally:

[Open the local app](http://localhost:5173/)
